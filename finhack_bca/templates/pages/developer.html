Integrasi mudah dilengkapi dengan tampilan riwayat transaksi

Konfirmasi manual
  1. Daftarkan website anda dengan menghubungi kami di daftarstore@example.com
  2. Buatlah button menggunakan logo SiDoKu dengan alamat url https://finhacks2016.pythonanywhere.com/konfirmasi/?store=(ID_STORE)&amount=(JUMLAH_PEMBAYARAN)&remarks=(INFO_TAMBAHAN)
  3. Cek status transaksi pada akun anda
konfirmasi otomatis
  1. Daftarkan website anda dengan menghubungi kami di daftarstore@example.com. Tentukan private key untuk me-hash remarks yang akan digunakan oleh SiDoKu dan website anda.
  2. Buatlah button menggunakan logo SiDoKu dengan alamat url https://finhacks2016.pythonanywhere.com/konfirmasi/?store=(ID_STORE)&amount=(JUMLAH_PEMBAYARAN)&remarks=(INFO_TAMBAHAN)
  3. Implementasi backend anda untuk menerima URL dengan hasil hash remarks menggunakan PKBDF2 dan private key anda. Pastikan remarks tiap transaksi berbeda.
  4. Setelah pelanggan melakukan pembayaran, maka SiDoKu akan me-redirect ke URL anda.
